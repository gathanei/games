Classic Games Implementation of classic games

Here you can play some classic games.

Snake

Snake the classic game. You are steering the snake, use the arrow keys to change direction. To grow you need to eat the apples. But don't eat yourself!

Play!

Mastermind

Are you a mastermind? Try to guess the code of colors. You have 12 guesses. For each correct color and position you get a black pin. For correct colors but wrong positions you get a white pin. The game is over if you use up all 12 trials.

Play!
